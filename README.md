Finals Exam Delivery Program

# Project overview
The program takes customer details and assign the order to driver then the driver send the item to the customer.
---
features: "Person" class and it's child class has introduce function to say hi and the "Customer" class has place_order function to place order and the "Driver" class has deliver function to print the text that what item is delivering to who then it change the status of that order to "delivered" and "DeliveryOrder" class that has assign driver function to assign driver into variable and summary function to show the detail of the order
